COFFE TIME

CoffeeCols is a fun, creative magazine style template that uses equal boxes on the home page to make content stand out.The boxes line themselves up and space themselves out regardless of how many you have, so you never have to deal with that.Because the style of the home page can't really be used on inner pages, this template also comes with a template for the inner pages.Two for the price of one.
The rounded corners, coffee-inspired tones, and content boxes really make this template different.Of course, it relies upon HTML5, CSS3, and jquery, so the template degrades nicely for browsers that are still playing catchup, but those users will see a less-fancy, though still functional site.
